POS Product Sync REST API

Real-time Product Sync | WooCommerce v3 Compatible

Base URL

https://yourdomain.com/wp-json/pos/v1

Authentication

headers: { 'Authorization': 'Bearer YOUR_JWT_TOKEN' }
Invalid token → 401/403

Endpoints

MethodEndpointDescription
GET/productsGet all synced
GET/product/{id}Get by POS ID
POST/productCreate
PUT/product/{id}Update by POS ID
DELETE/product/{id}Delete by POS ID

1. Get All POS Products

GET /products
fetch('https://yourdomain.com/wp-json/pos/v1/products', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
})
.then(r => r.json())
.then(d => console.log(d));

Possible Errors

401: { "status": "error", "message": "Authorization header missing" }
403: { "status": "error", "message": "Invalid token" }
500: { "status": "error", "message": "WooCommerce API error" }

2. Get Single Product

GET /product/123
fetch('https://yourdomain.com/wp-json/pos/v1/product/123', {
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
})
.then(r => r.json())
.then(d => console.log(d));

Possible Errors

404: { "status": "false", "message": "Product not found" }
401/403: Invalid token

3. Create Product

POST /product
fetch('https://yourdomain.com/wp-json/pos/v1/product', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    pos_id: 999,
    data: { name: "T-Shirt", regular_price: "29.99" }
  })
})
.then(r => r.json())
.then(d => console.log(d));

Possible Errors

400: { "status": "error", "message": "POS id is required" }
400: { "status": "error", "message": "Product data is required" }
409: { "status": "error", "message": "Product already exists" }
500: { "status": "error", "message": "Failed to insert POS record" }
WooCommerce: { "code": "woocommerce_rest_product_invalid", "message": "..." }

4. Update Product

PUT /product/999
fetch('https://yourdomain.com/wp-json/pos/v1/product/999', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    data: { regular_price: "34.99" }
  })
})
.then(r => r.json())
.then(d => console.log(d));

Possible Errors

400: { "status": "error", "message": "POS id is required" }
400: { "status": "error", "message": "Product data is required" }
404: { "status": "error", "message": "Product not found" }
WooCommerce: { "code": "woocommerce_rest_cannot_edit", "message": "..." }

5. Delete Product

DELETE /product/999
fetch('https://yourdomain.com/wp-json/pos/v1/product/999', {
  method: 'DELETE',
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
})
.then(r => r.json())
.then(d => console.log(d));

Possible Errors

400: { "status": "error", "message": "POS id is required" }
404: { "status": "error", "message": "Product not found" }
500: { "status": "error", "message": "Failed to delete from sync table" }
